    How to share a document: when you upload, pick the tag that matches the minimum level you want to require. Tag L1 → everyone sees it. Tag L → only L+ members see it. You can only tag up to your own level.

    Personal docs are never shared — visible only to you, regardless of anyone else's clearance level.

    Level names come from this workspace's clearance_scheme config. Only the number is stored and indexed, so renaming a level never re-embeds anything.

Your groups Phase 2

security eng-space confluence platform-team git

Groups narrow access — they never widen it. A document tagged L2 + security is visible only to L2-or-higher members who are also in security. Both checks must pass.

Rank does not grant a group. An owner or admin who isn't in security cannot read its documents — they can only administer who belongs to it.

Mirrored groups (confluence, git) are managed in the source system and are read-only here.
